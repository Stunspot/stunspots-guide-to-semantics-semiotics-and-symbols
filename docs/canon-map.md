# Canon Map — Stunspot's Guide to Semantics, Semiotics, and Symbols

This canon maps **Semantic-Semiotic Intelligence (SSI)** as an operational discipline for models: how meaning is represented, how signs function, how claims become warranted, how context changes interpretation, how symbols accumulate social force, and how assistants can govern semantic failure at runtime.

The repository contains **14 source reports** grouped into **4 conceptual volumes**. The source reports are the canonical individual units and live in `knowledge-packs/by-report/`.

---

## Volume 1 — Foundations of Representation, Meaning, and Evaluation

Volume 1 establishes the core machinery of meaning: map/territory discipline, sign-relations, linguistic structure, conceptual frames, and epistemic warrant. It is the best starting point for readers or models that need the canon’s primitives before using later diagnostic or governance layers.

| Code | Report | Function in the Canon |
|---|---|---|
| A | [Reality, Representation & Abstraction](https://github.com/Stunspot/stunspots-guide-to-semantics-semiotics-and-symbols/blob/main/knowledge-packs/by-report/a-reality-representation-and-abstraction.md) | Establishes the semantic constitution: territory vs. map, non-identity, non-allness, indexing, dating, abstraction ladders, and representational failure modes. |
| B | [Sign-Relations, Codes & Semiosis](https://github.com/Stunspot/stunspots-guide-to-semantics-semiotics-and-symbols/blob/main/knowledge-packs/by-report/b-sign-relations-codes-and-semiosis.md) | Defines signs as relation-events among sign-vehicle, object, interpretant, code, and context; introduces semiosis, icon/index/symbol distinctions, and code systems. |
| C | [Linguistic Structure & Compositional Meaning](https://github.com/Stunspot/stunspots-guide-to-semantics-semiotics-and-symbols/blob/main/knowledge-packs/by-report/c-linguistic-structure-and-compositional-meaning.md) | Moves from signs to language structure: syntax, compositionality, lexical meaning, reference-bearing expressions, and how structure constrains interpretation. |
| D | [Concepts, Categories & Frames](https://github.com/Stunspot/stunspots-guide-to-semantics-semiotics-and-symbols/blob/main/knowledge-packs/by-report/d-concepts-categories-and-frames.md) | Explains how meanings cluster into concepts, prototypes, categories, frames, schemas, and interpretive expectations. |
| E | [Reference, Grounding & Epistemic Validity — The Architecture of Warranted Semantic Action](https://github.com/Stunspot/stunspots-guide-to-semantics-semiotics-and-symbols/blob/main/knowledge-packs/by-report/e-reference-grounding-and-epistemic-validity.md) | Converts meaning into warranted use: claim, referent, truth condition, evidence, source relation, grounding path, confidence, uncertainty, and permissible action. |

## Volume 2 — Operating Domains of Meaning Construction

Volume 2 shows meaning operating in context: pragmatic inference, metaphor, discourse, narrative, culture, ritual, memes, multimodal signs, and interfaces. It is the most useful volume for assistants that must interpret human communication rather than merely summarize documents.

| Code | Report | Function in the Canon |
|---|---|---|
| F | [Context, Intention & Pragmatic Inference](https://github.com/Stunspot/stunspots-guide-to-semantics-semiotics-and-symbols/blob/main/knowledge-packs/by-report/f-context-intention-and-pragmatic-inference.md) | Models how context, speaker intent, implicature, deixis, common ground, and pragmatic cues change meaning. |
| G | [Metaphor, Embodiment & Cognitive Models](https://github.com/Stunspot/stunspots-guide-to-semantics-semiotics-and-symbols/blob/main/knowledge-packs/by-report/g-metaphor-embodiment-and-cognitive-models.md) | Explains metaphor as structured conceptual mapping, including embodied schemas and model-transfer risks. |
| H | [Discourse, Narrative & Social Reality](https://github.com/Stunspot/stunspots-guide-to-semantics-semiotics-and-symbols/blob/main/knowledge-packs/by-report/h-discourse-narrative-and-social-reality.md) | Treats language as social action: discourse roles, narrative framing, institutional meaning, status, legitimacy, and shared reality formation. |
| I | [Symbolic Culture, Myth, Ritual & Memetics](https://github.com/Stunspot/stunspots-guide-to-semantics-semiotics-and-symbols/blob/main/knowledge-packs/by-report/i-symbolic-culture-myth-ritual-and-memetics.md) | Maps cultural symbols, mythic structures, ritual action, memetic propagation, and the social charge attached to recurring symbolic forms. |
| J | [Multimodal Signs & Interface Symbolics](https://github.com/Stunspot/stunspots-guide-to-semantics-semiotics-and-symbols/blob/main/knowledge-packs/by-report/j-multimodal-signs-and-interface-symbolics.md) | Extends SSI into images, layout, UI, diagrams, icons, affordances, visual hierarchy, and cross-modal meaning. |

## Volume 3 — Constraint, Instability, and Translation Layers

Volume 3 covers meaning under stress: ambiguous words, drifting categories, transfer between domains, and cross-cultural variance. Use it when an assistant must preserve nuance across unstable terminology or different interpretive communities.

| Code | Report | Function in the Canon |
|---|---|---|
| K | [Ambiguity, Polysemy & Semantic Drift](https://github.com/Stunspot/stunspots-guide-to-semantics-semiotics-and-symbols/blob/main/knowledge-packs/by-report/k-ambiguity-polysemy-and-semantic-drift.md) | Diagnoses sense ambiguity, polysemy, category instability, meaning drift, stale reference, and lexical misbinding. |
| L | [Translation, Domain Transfer & Cross-Cultural Variance](https://github.com/Stunspot/stunspots-guide-to-semantics-semiotics-and-symbols/blob/main/knowledge-packs/by-report/l-translation-domain-transfer-and-cross-cultural-variance.md) | Governs translation across languages, domains, cultures, disciplines, interfaces, and symbolic systems without false equivalence. |

Volume 4 turns the canon into a runtime discipline. It identifies semantic pathologies and then defines infrastructure for active metadata, semantic routing, provenance, interpretive lenses, and assistant governance.

| Code | Report | Function in the Canon |
|---|---|---|

| N | [Knowledge Infrastructure, Artifacts & Assistant Governance — The Execution Layer of the Semantic-Semiotic Intelligence Canon](https://github.com/Stunspot/stunspots-guide-to-semantics-semiotics-and-symbols/blob/main/knowledge-packs/by-report/n-knowledge-infrastructure-artifacts-and-assistant-governance.md) | Defines the execution layer: ontological minimalism, active metadata, reusable lenses, semantic routing, provenance, citation faithfulness, and governance protocols. |
