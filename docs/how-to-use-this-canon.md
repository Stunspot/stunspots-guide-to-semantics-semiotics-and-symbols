# How to Use This Canon

For most AI/RAG systems, start with the **compiled packs** in `knowledge-packs/compiled-packs/`.

| Format | Path | Best For |
|---|---|---|
| Source reports | `knowledge-packs/by-report/` | precise retrieval, selective upload, citation, editing |
| Compiled packs | `knowledge-packs/compiled-packs/` | recommended default for most systems |
| Omnibus | `knowledge-packs/omnibus/` | one-file import, local archive, strong long-context/RAG systems |
