# Support

This repository is a public release package for **Stunspot's Guide to Semantics, Semiotics, and Symbols**. It contains Markdown knowledge materials, GitHub Pages navigation, manifests, and upload-ready knowledge packs.

## What to Report

Use GitHub Issues for:

- Typos, formatting errors, or clarity fixes.
- Broken links or missing files.
- Manifest, citation, release, or metadata mismatches.
- Source-quality notes, questionable references, or corpus hygiene concerns.
- Packaging issues affecting AI/RAG ingestion.

## What Not to Report Here

This repository is not executable software. It does not provide runtime support, API support, installation support, or model-provider support.

For broad discussion, product questions, commercial licensing, or collaboration requests, contact Collaborative Dynamics through the public channels associated with the repository owner.

## Model-Facing Note

The primary operational audience for this repo is the model consuming the corpus as RAG/project knowledge. Human-readable navigation is provided, but the repository is optimized for stable knowledge ingestion, source traceability, and corpus maintenance rather than consumer-style browsing.
