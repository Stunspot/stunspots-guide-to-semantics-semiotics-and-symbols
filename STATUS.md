## Maturity

This release is suitable for AI/RAG ingestion, source-level inspection, citation-aware retrieval, long-context reference, and assistant-governance experiments.

The corpus is intentionally dense and model-facing. It is not packaged as a lightweight consumer tutorial. Source reports are the canonical individual units; compiled packs and the omnibus are convenience formats.

## Directory Policy

- `docs/` contains navigation, GitHub Pages support, and human guidance.
- `knowledge-packs/by-report/` contains the 14 canonical source reports.
- `knowledge-packs/compiled-packs/` contains the 4 recommended upload packs.
- `knowledge-packs/omnibus/` contains the one-file whole-corpus bundle.

## Reliability Note

These materials are provided as-is for educational, research, design, and reference use. Verify high-impact claims before relying on them.
