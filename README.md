**A model-facing canon for semantics, semiotics, symbols, meaning, reference, and interpretation.**

## Corpus Shape

- **14 source reports** in `knowledge-packs/by-report/`
- **4 compiled packs** in `knowledge-packs/compiled-packs/`
- **1 omnibus file(s)** in `knowledge-packs/omnibus/`

`docs/` contains navigation and GitHub Pages files. Individual reports live only in `knowledge-packs/by-report/`.

